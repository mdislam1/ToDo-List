Function Explanations

This section explains functionalities that are not self explanitory


When a category is deleted, we call a stored procedure "removeCategory(category_id)", that procedure sets all tasks that are categorized as the category being deleted to null. After that, we delete the category

When a task is marked as complete, we call a stored procedure "markAsComplete(task_id)" that procedure stores the date of when the task is marked as complete into the tasks table and it changes the completion status of the task to completed
